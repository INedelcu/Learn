Interactive companion pages for the C++ Monte Carlo tutorial series.

Each page below explains one sample from the C++ series — the probability and statistics behind Monte Carlo integration and physically-based path tracing — with the math typeset, the code annotated, and interactive figures you can play with directly in the browser. Read them in order; each builds on the last.

The whole series is, ultimately, about one formula — estimating an integral as the average of (integrand)/(sampling density) — and about understanding and shrinking its error. These explainers develop that story one idea at a time.
